The following tables list the members exposed by Settings.Style.

Public Properties

  Name Description
Completed This style is used for all Squares of which there are exactly 9 on the Board, whether they are in the right places or not.
CompletedBlock This style is used for all Squares in a completed Block, whether or not the values are correct.
CompletedOnSelectionBar This style is used for the Squares on the SelectionBar of which there are exactly 9 on the Board, whether they are in the right places or not.
CompletedRowOrColumn This style is used for all Squares in a complted Row or Column, whether or not the values are correct.
EverythingElse This style is used for all Squares that don't fit any other category.
EverythingElseOnSelectionBar This style is used for all Squares on the SelectionBar that don't fit any other category.
EverythingElsePencilMarks This style is used for all PencilMarks that don't fit any other category.
Given This style is used for all permanent Squares that were part of the original game.
Selected This style is used for all Squares whose Value matches that of the SelectionBar's SelectedSquare.
SelectedPencilMark This style is used for all PencilMarks whose Value matches that of the SelectionBar's SelectedSquare.
ToggleSelectedStyleOnRepeatedTaps Specifies whether or not multiple taps on the SelectionBar activates / deactivates the Selected style settings.

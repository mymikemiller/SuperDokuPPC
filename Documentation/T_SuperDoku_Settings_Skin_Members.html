Settings.Skin Members

Settings pertaining to the graphical Skin used in the game. Change the Skin via SetSkin(). This is a static class (Module in Visual Basic)static classModule and so cannot be inherited or instantiated.


The following tables list the members exposed by Settings.Skin.

Public Properties

  Name Description
BackgroundRect Gets the Rectangle containing the Background in the skin
Bitmap Gets the Skin bitmap
BoardLength Gets the pixel length of an unscaled Board created from the Squares in the skin. Because some squares overlap, this is slightly less than SkinSquareLength * 9.
FileName Gets the filename of the Skin. Change via that SetSkin method.
GutterLength Gets the number of pixels between each Square in the skin
IdealBoardLengths A list of Board lengths (i.e. widths/heights) that cause little distortion when resized from the BoardLength, i.e. half, 2x, etc.
OverlapLength Gets the number of pixels of the border on each Square in the skin
PencilMarkLongLength Gets the pixel length of the long edge of a PencilMark in the skin
PencilMarkShortLength Gets the pixel length of the short edge of a PencilMark in the skin
SquareLength Gets the pixel length of the Squares in the skin
